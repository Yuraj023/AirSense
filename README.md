🚀 Built My Own Smart Air Quality Monitoring System – *AirSense*

Air pollution is a serious problem, especially in urban environments. I wanted to go beyond theory and build a real-world solution that can monitor and respond to air quality in real time.

So I developed **AirSense** – a compact, IoT-based air quality monitoring and alert system.

🔧 **What AirSense Does:**
• Monitors harmful gases, temperature, and humidity in real time
• Detects fine particulate matter (PM2.5)
• Sends instant email alerts when air quality crosses safe thresholds
• Streams live data to a cloud dashboard

🧠 **Tech Stack & Components:**
• ESP32 (WiFi-enabled microcontroller)
• MQ-135 (air quality / gas sensor)
• DHT11 (temperature & humidity sensor)
• GP2Y1010AU0F (PM2.5 dust sensor)
• Battery-powered system with step-down (LMS module)
• Cloud integration + real-time dashboard
• Email alert automation system

⚙️ **Key Features:**
✔ Real-time environmental monitoring
✔ Threshold-based alert system (email notifications)
✔ Cloud-connected IoT system
✔ Compact and energy-efficient design

💡 **What I Learned:**
• Integrating hardware with cloud platforms
• Handling real-time sensor data
• Building alert systems based on conditions
• End-to-end IoT system design (device → cloud → user)

This project helped me understand how data from the physical world can be transformed into actionable insights using IoT + software.
